# CSC281 Group Project: Expense Tracker

A Vue.js application to track daily expenses with data visualization.

## Features
- Add/Remove Expenses: Easily manage your spending records.
- Visual Analytics: Real-time Pie Chart showing spending by category.
- Total Calculator: Automatically sums up total expenditure.
- Persistent Data: Saves your data even if you refresh the browser.
- Dark Mode: Adapts to your device's theme settings.
